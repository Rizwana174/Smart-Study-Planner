# 📚 Smart Study Planner

A simple and responsive web application that helps students organize their study schedules, manage tasks, and track learning progress efficiently.

## 🚀 Live Demo

🔗 https://rizwana174.github.io/Smart-Study-Planner/

## 📂 GitHub Repository

🔗 https://github.com/Rizwana174/Smart-Study-Planner

---

## ✨ Features

* Add study subjects and topics
* Set deadlines for study tasks
* Organize learning activities efficiently
* Track study progress
* Responsive and user-friendly interface
* Works directly in the browser without installation

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages

---

## 📸 Project Preview

### Home Page

* Clean dashboard for managing study tasks
* Easy-to-use interface for students

### Task Management

* Add subjects and topics
* Set due dates
* View all study tasks in one place

### Progress Tracking

* Monitor completed tasks
* Track overall study progress

---

## 🎯 Project Objective

The goal of this project is to help students maintain consistency in their studies by providing a simple platform for planning, organizing, and tracking academic tasks.
